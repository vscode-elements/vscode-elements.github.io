The unique feature of the Table component, as compared to the standard table element, is its fixed header and resizable columns.

## Scrollable body

If the table is assigned a fixed height and the content exceeds the table's height, the content will become scrollable.

<Demo>
  <ScrollableExample />
  <Fragment slot="html">
    ```html

      <vscode-table-header slot="header">
        <vscode-table-header-cell>First name</vscode-table-header-cell>
    ```

## Responsive layout

The table switch to compact layout mode when its narrower than the set breakpoint.

<Demo>
  <ResponsiveExample />
  <Fragment slot="html">
