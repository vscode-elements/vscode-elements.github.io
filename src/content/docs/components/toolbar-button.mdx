## Basic example

In icon-only mode, a label for screen readers can be defined using the `label` attribute.

<Demo>
  <vscode-toolbar-button icon="account" label="Account"></vscode-toolbar-button>
  <Fragment slot="html">
    ```html 'label="Account"'
    <vscode-toolbar-button
      icon="account"
      label="Account"
    ></vscode-toolbar-button>
    ```
  </Fragment>
</Demo>

<Demo>

    <vscode-toolbar-button

  <Fragment slot="js">
